# 文件资源夹说明

本文件夹用于存放课程资源文件，按分类组织。

## 文件夹结构

```
files/
├── math/          # 数学相关资料
├── cs/            # 计算机相关资料
├── physics/       # 物理相关资料
├── language/      # 语言相关资料
├── economics/     # 经济相关资料
└── humanities/    # 人文社科相关资料
```

## 文件命名规范

为了确保系统正确识别和关联资源，请按以下格式命名文件：

### 数学 (math/)
- `高等数学笔记.pdf`
- `线性代数资料.pdf`
- `概率论与数理统计.pdf`

### 计算机 (cs/)
- `C语言程序设计.pdf`
- `数据结构与算法.pdf`
- `Python入门指南.pdf`
- `数据库实践讲义.pdf` ← 例如数据库课程使用此格式

### 物理 (physics/)
- `大学物理公式汇总.pdf`
- `电路分析基础.pdf`

### 语言 (language/)
- `英语四六级词汇.pdf`
- `大学英语教程答案.pdf`

### 经济 (economics/)
- `经济学原理笔记.pdf`

### 人文 (humanities/)
- `心理学概论.pdf`
- `中国近代史纲要.pdf`

## 数据关联说明

文件路径与 `data.js` 中的资源数据对应：

```javascript
{
    id: "RES_000003",
    title: "C语言程序设计教程",
    filePath: "files/cs/C语言程序设计.pdf",  // ← 文件必须在此路径
    uploader: "王同学",                        // ← 上传者标识
    ...
}
```

## 上传流程

1. 将文件放入对应分类文件夹（如 `files/cs/`）
2. 按照上述命名规范重命名文件
3. 在管理后台或通过上传页面添加资源记录

## 注意

- 删除占位文件.txt即可
- 支持格式：PDF、Word、PPT、TXT 等常见格式
