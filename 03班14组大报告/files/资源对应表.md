## 添加新资源的方法

1. **放置文件**到对应分类文件夹
   ```
   files/cs/数据库实践讲义.pdf
   ```

2. **更新 data.js** 中的 `_initSampleData` 数组：
   ```javascript
   { 
       id: "RES_000013", 
       title: "数据库实践讲义", 
       category: "cs", 
       filePath: "files/cs/数据库实践讲义.pdf",
       fileName: "数据库实践讲义.pdf",
       uploader: "廖同学",    // ← 上传者用户名
       uploaderId: "USR_000001",  // ← 上传者ID
       // ... 其他字段
   }
   ```

3. **更新本表格**记录新资源
